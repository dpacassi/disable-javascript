Please note that the host name being entered has to match completely in order to work.
Entering e.g. youtube.com won't work as the site runs as www.youtube.com, which is technically another host.

You can, however, insert a base domain such as youtube.com and include it's sub-domains by checking the appropriate checkbox.
But be careful when toggling the JS state via icon or contextual menu then as it will remove the parent entry and not create a custom one for the sub-domain!
Complete inheritance is not supported yet!
