About Disable JavaScript

Note: This web extension is completely free and open source!
Please consider making a small donation or write a review on addons.mozilla.org and/or on the chrome web store, thank you!

First off, thank you for using this web extension!

If you find any bugs or have any suggestions/ideas for this web extension, please create a new issue on the projects GitHub page.
I've received a few reports where this web extension is not always working. However, I was not able to reproduce this bug as I didn't get enough information to do so.

If you happen to find this web extension not working on a particular machine/browser/website, please provide me details like browser, browser version, operating system and website url so I can check the web extension for any bugs. Read more about this on GitHub.

Thank you for your support!

What's new

Fixed two bugs that resulted in wrongly showing <noscript> tags even though JavaScript was enabled.

2.3.0

Added the ability to edit existing hosts. Fixed a bug when checking for valid hosts.

Shortcuts

Important: You can enable or disable shortcuts in the settings page.

  • Alt + Shift + 1
    Toggle JavaScript state
  • Alt + Shift + 2
    Open Disable JavaScript settings

Changelog

  • 2.3.1 (September 27th, 2018)
    Fixed two bugs that resulted in wrongly showing <noscript> tags even though JavaScript was enabled.
  • 2.3.0 (September 9th, 2018)
    Added the ability to edit existing hosts. Fixed a bug when checking for valid hosts.
  • 2.2.3 (July 1st, 2018)
    Remove source SASS and JS files from the web extension package script as they are not needed in the final environment.
  • 2.2.2 (July 1st, 2018)
    Fixed a bug which caused to falsely display <noscript> tags and fixed another bug that allowed to block empty url's.
  • 2.2.1 (March 31st, 2018)
    Fixed a caching bug when disabling/enabling JS.
  • 2.2.0 (March 22nd, 2018)
    Removed reloading all tabs after web extension/browser installation/update.
    Added the ability to add base domains that automatically disable JS for any sub domains.
  • 2.1.0 (March 18th, 2018)
    Added new settings like enabling or disabling shortcuts and context menu items.
  • 2.0.1 (February 25th, 2018)
    Fixed a bug that broke the web extension when installing or updating on Google Chrome.
  • 2.0.0 (February 25th, 2018)
    Added a settings page, add an about page. Also introduced shortcut commands.
  • 1.2.0 (December 10th, 2017)
    Added support for Microsoft Edge (has to be added manually to Microsoft Edge [deprecated by 2.0.0]).
  • 1.1.0 (November 25th, 2017)
    Added support for Firefox for Android.
  • 1.0.1 (November 21st, 2017)
    Reloading active tabs after web extension installation.
  • 1.0.0 (November 20th, 2017)
    Initial release.
